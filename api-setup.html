如何配置到毅鱼漫剧

打开应用，三步完成

1

打开「设置 → AI 服务」

在毅鱼漫剧侧边栏进入设置页面，点击「AI 服务」标签页。

2

按类型添加配置

分别点击「文本」「图片」「视频」「音频」卡片的「添加」按钮，填入上方对应的 Provider / Base URL / API Key / 模型。

3

测试 & 启用

点击「测试」按钮验证连通性，确认无误后勾选启用开关，即可开始AI创作。

没有自己的 Key？

毅鱼已预置默认 Key，打开 App 即可使用，无需额外配置。
